## 簡介
為前端練習作品，目的為正式上線並串上籃新金流。
+ 前台: 所有商品、商品分類、以及商品細項展示，購物車功能、下單、結帳資訊、管理員登入等。
+ 後台: 商品管理（新增/修改/刪除）、訂單管理、優惠券管理。

### 主要練習
+ 前台 Bootstrap 自定義元件設計
+ RWD
+ VueCli3.0
+ VueRouter
+ Vuex
+ Webpack
+ axios 操作 API
+ Vue 自定義元件
